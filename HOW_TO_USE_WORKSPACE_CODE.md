## Troubleshooting

### Problem: Code says "Invalid workspace code"

**Solution**:

- Make sure the code is exactly 8 characters
- Check that you copied it correctly (should be uppercase letters and numbers)
- Verify the code hasn't been mistyped

### Problem: Code says "Member user not found"

**Solution**:

- The person joining must have a valid account
- First signup/login on their own account
- Then use Join Workspace with the owner's code

### Problem: "You are already the workspace owner"

**Solution**:

- You can't join your own workspace
- This error means you're trying to join your own code

### Problem: Join Workspace tab not visible

**Solution**:

- Refresh the page (F5 or Ctrl+R)
- Make sure you have the latest version of the code
- Check that you're looking at the tab buttons at the bottom of the right panel

### Problem: Code changes every login

**Solution**:

- This should NOT happen
- If it does, try clearing browser cache and relogging
- The code should always be the same for the same email

## FAQs

**Q: Can I use the same code multiple times?**
A: Yes! The code never changes, so multiple team members can use the same code.

**Q: What if I lose the code?**
A: The workspace owner can always see their code in the purple box and share it again.

**Q: Can I disable or delete a code?**
A: Currently no - codes are permanent. Future versions may add code management.

**Q: Does the code work across different devices?**
A: Yes! You can share the code via any method and use it on any device.

**Q: What if someone uses my code without permission?**
A: They will be added as a team member to your workspace. In future versions, you'll be able to remove members.
